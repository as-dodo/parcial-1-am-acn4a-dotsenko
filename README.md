# parcial-1-am-acn4a-dotsenko

## Descripción breve
Aplicación Android simple para gestionar "rachas" (streaks). Permite crear nuevas rachas, marcarlas como completadas, ver un resumen diario y navegar entre las secciones principales (Inicio, Calendario, Amigos, Perfil).

## Funcionalidades principales
- Crear y listar rachas.
- Marcar rachas como completadas y actualizar la duración.
- Resumen diario de rachas completadas.
- Navegación inferior con las secciones principales.
- Recursos centralizados en `app/src/main/res/values` (strings, colors, dimens).
